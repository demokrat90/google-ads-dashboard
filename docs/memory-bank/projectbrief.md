# Project Brief: Ads Dashboard

## Цель проекта
Веб-приложение для мониторинга рекламных кампаний Google Ads с интеграцией amoCRM и базой застройщиков недвижимости.

## Основные функции

### Страница 1: Google Ads + amoCRM (`/ads`)
- Таблица кампаний и групп объявлений
- Расход по каждой кампании/группе
- Лиды из amoCRM (по UTM-меткам)
- Квалифицированные лиды (с тегом "At work")
- Расчёт CPL (Cost Per Lead) и CPQL (Cost Per Qualified Lead)
- Период: неделя суббота→пятница
- История недель внизу (обновляется 8 недель для квал. лидов)

### Страница 2: Застройщики (`/developers`)
- Выбор застройщика из списка (496 застройщиков)
- Проекты застройщика (2291 проект)
- Количество юнитов в продаже (35211 юнитов всего)
- Фильтр: все / только с юнитами
- Связь с рекламой по названию кампании
- Настройка избранных застройщиков

## Целевая аудитория
Команда 2-5 человек (маркетинг/продажи недвижимости)

## Хостинг

### Текущий (Express)
- VPS Hostinger
- Node.js + Express + EJS
- SQLite локально для кеша

### Планируемый (Next.js)
- **Vercel** — хостинг приложения
- **MySQL Hostinger** — две базы данных:
  - Основная (существующая) — застройщики, проекты, юниты
  - `ads_dashboard` (новая) — Google Ads, лиды amoCRM, история
